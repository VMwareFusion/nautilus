## vctl rmi

Remove one or more container images.

### Synopsis

Delete all images or delete by the given container image(s).

```
vctl rmi [OPTIONS] ([IMAGE...]|--all)
```

### Options

```
  -a, --all     Delete all images
  -f, --force   Force removal of image regardless of its status
  -h, --help    Help for rmi
```
